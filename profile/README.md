# Gambling Golfer

A mobile-first golf gaming platform for tracking scores, handicaps, and friendly wagers.

## Repositories

| Repository | Description |
|------------|-------------|
| [backend](https://github.com/Gambling-Golfer/backend) | Node.js/Express API server |
| [iOS](https://github.com/Gambling-Golfer/iOS) | iOS app (SwiftUI) |

## Tech Stack

- **Backend**: Node.js, Express, TypeScript, PostgreSQL
- **iOS**: Swift, SwiftUI, MVVM, Swift Package Manager
- **Infrastructure**: Azure (see private infra repo)

## Contributing

Each repository has its own contributing guidelines and Copilot instructions. See the individual repos for details.
